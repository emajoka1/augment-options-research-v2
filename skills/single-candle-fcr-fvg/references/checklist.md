# FCR + FVG Go/No-Go Checklist

- [ ] First 5-minute candle closed
- [ ] FCR high and low marked
- [ ] Clean break and close outside FCR
- [ ] M1 displacement confirmed
- [ ] Valid FVG created in break direction
- [ ] Entry at FVG retrace (not chased)
- [ ] Stop placed beyond invalidation
- [ ] RR >= 1:2 to target
- [ ] No major news in next 10-15 min
- [ ] Spread/liquidity acceptable
- [ ] Risk size within 0.25%-0.75%

If critical checks fail, PASS.
