# Pre-Trade Checklist (Sweep + iFVG)

- [ ] Asia/London high/low marked
- [ ] Sweep occurred at a session liquidity level
- [ ] Close back inside range (no immediate continuation)
- [ ] 1-3m displacement confirmed
- [ ] Opposite-direction iFVG formed
- [ ] Entry at iFVG retrace planned
- [ ] Stop beyond sweep extreme
- [ ] TP1=1R and TP2=2R (or opposing liquidity)
- [ ] R:R >= 1:2 available
- [ ] No high-impact news in next 10-15 min
- [ ] Risk size computed (0.25%-0.75%)
- [ ] Invalidation condition defined

If any critical item fails: PASS.
